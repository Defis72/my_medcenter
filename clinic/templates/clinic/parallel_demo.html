Используется asyncio.gather() для конкурентного выполнения корутин. Asyncio работает в одном потоке, но обрабатывает несколько задач через event loop.

Пояснение

Threading подходит для I/O-задач. Потоки разделяют память и используют Lock для безопасного доступа. GIL не позволяет параллельно выполнять Python-код, но при ожидании I/O потоки работают конкурентно.

Asyncio — кооперативная многозадачность в одном потоке. Функция сама решает, когда отдать управление через await. Идеально для большого числа лёгких сетевых задач.
